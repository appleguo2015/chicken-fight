# Chicken Fight
## key
- `a d`/`h k`: move left/right
- ` `: squat
- `q`: quit
## entity
- `chicken`: move left/right, squat
```
_%-  <- chicken
##
^^
```
- `bullet`: move left
```
*   <- bullet
```
- `be hit chicken`: dieeeeeee
```
**  <- dead chicken
##
^^
```
## how to run
```bash
git clone https://github.com/appleguo2015/chicken_fight.git
cd chicken_fight/cmake-build-debug
cmake ..
./chicken_fight
```
